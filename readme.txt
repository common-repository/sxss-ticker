== Description ==

Make a simple news ticker (table, list, marquee)

More informations on the official plugin page: [http://sxss.info/wordpress-ticker/](http://sxss.info/wordpress-ticker/ "sxss.info")

== Installation ==

Upload the sxss Ticker plugin to your blog, activate it, go to 'Settings > sxss Ticker'.
